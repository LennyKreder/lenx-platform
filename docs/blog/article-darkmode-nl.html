Voor de iPad planners had ik oorspronkelijk 5 donkere modus kleurthema's; dark mode blue, dark mode brown, dark mode olive, dark mode purple en dark mode rose. Allemaal met een net iets andere sfeer. Maar vandaag dacht ik, waarom heb ik niet de puurste dark mode weergave die mogelijk is? Zwart-wit. Puur en simpel. Als ik code schrijf, vind ik het kleurenschema van mijn code editor geweldig. Dus heb ik 2 nieuwe kleurthema's toegevoegd aan de iPad minimalist planners; dark mode essentials en dark mode terminal.

Ik werk graag in dark mode, het heeft iets rustgevends. Ook komen kleuren feller over dan in light mode. Het contrast is fijn.

Dark mode essentials is precies wat het klinkt. Zwarte achtergrond, witte tekst, geen afleiding. Het is het planner-equivalent van een leeg notitieboek op een donker bureau. Alleen jij en je gedachten.

Dark mode terminal is voor de nerds. En dat zeg ik met liefde, want ik ben er zelf een. Het is geïnspireerd op de code editor waar ik de hele dag naar staar — witte tekst op een zwarte achtergrond, met accenten in oranje, blauw en groen. Als je ooit urenlang in een IDE hebt gezeten, voelt dit als thuiskomen. Als dat niet zo is — het ziet er alsnog gaaf uit.

Ik denk dat mensen onderschatten hoeveel de kleur van je planner beïnvloedt hoe je hem gebruikt. Een felwit scherm om 23:00 als je je weekoverzicht doet? Dat is niet rustig, dat is een zaklamp in je gezicht. Dark mode is niet alleen een esthetische keuze. Het is een comfortding. Zeker op een iPad waar je letterlijk naar een lichtbron zit te staren.

De light mode versies gaan nergens heen, trouwens. Sommige mensen plannen 's ochtends met hun koffie en zonlicht. Daar is light mode voor. Maar als je een avondplanner bent, of gewoon de voorkeur geeft aan iets rustiger voor je ogen — nu heb je zeven dark themes om uit te kiezen.
